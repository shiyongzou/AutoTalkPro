# AutoTalk Pro 新手入门教程

---

## 1. 安装与首次启动

### 下载安装包

- **国内下载**：https://gitee.com/zoushiyong_admin/AutoTalkPro/releases
- **GitHub 下载**：https://github.com/shiyongzou/AutoTalkPro/releases

下载对应系统的 zip 包：
- macOS：`AutoTalkPro-macOS.zip`
- Windows：`AutoTalkPro-Windows.zip`

### macOS 安装

1. 解压 zip 文件，得到 `Auto Talk Pro.app`
2. 拖入「应用程序」文件夹
3. 首次打开如提示"无法验证开发者"，前往「系统设置 → 隐私与安全性」点击"仍要打开"

### Windows 安装

1. 解压 zip 文件
2. 双击 `autotalk_pro.exe` 即可运行

### 首次启动

首次启动时应用会自动下载 Node.js 运行环境（约 60MB），请保持网络畅通，等待 1-2 分钟。下载完成后自动进入登录页面。

## 2. 登录聊天渠道

### 方案 A：微信登录（推荐新手）

1. 点击「微信」
2. 等待二维码出现（约 5-10 秒）
3. 用手机微信扫一扫
4. 手机确认登录
5. 看到"登录成功"后点击「进入平台」

> ⚠️ 建议使用微信小号测试，微信网页协议存在封号风险。

### 方案 B：Telegram 登录

1. 点击「Telegram」
2. 输入手机号（含区号，如 `+8613800138000`）
3. 点击「获取验证码」
4. 在 Telegram 应用中查看验证码，输入
5. 登录成功

> 需要科学上网才能连接 Telegram。

### 方案 C：企业微信登录

1. 点击「企业微信」
2. 填写 Corp ID、Agent ID、Secret
3. 点击「验证并保存」

获取方式：登录企业微信管理后台（work.weixin.qq.com）→ 应用管理 → 查看应用详情。

## 3. 配置 AI 模型

进入主界面后，点击左侧「AI设置」。

### 第一步：选择平台

国内用户推荐：

| 平台 | 费用 | 特点 |
|------|------|------|
| **DeepSeek** | 极低 | 中文最佳，性价比最高 |
| **Ollama 本地** | 免费 | 数据不出本机 |
| **通义千问** | 低 | 阿里出品，稳定 |
| **豆包** | 低 | 响应最快 |

### 第二步：获取 API Key

以 DeepSeek 为例：

1. 打开 https://platform.deepseek.com/
2. 注册并登录
3. 进入「API Keys」页面
4. 点击「创建 API Key」
5. 复制 Key（如 `sk-xxxxxxxxxxxxxxxx`）
6. 粘贴到 AutoTalk Pro 的 API Key 输入框
7. 点击保存

> 使用 Ollama 本地模型无需 API Key。先安装 Ollama（https://ollama.com），运行 `ollama pull qwen2.5` 下载模型即可。

### 第三步：选择模型

- DeepSeek：选 `deepseek-chat`（日常）或 `DeepSeek-R1`（推理）
- 通义千问：选 `qwen-plus`（均衡）或 `qwen-turbo`（快速）

### 第四步：调节创意度（Temperature）

- **0.0 - 0.3**：严谨（医疗、法律等）
- **0.4 - 0.7**：均衡（推荐默认）
- **0.8 - 1.0**：创意（闲聊、写作）

## 4. 设置 AI 人设

点击左侧「人设」。

### 创建人设

点击「新建人设」，填写：

| 字段 | 示例（电商客服） | 示例（医生） |
|------|-----------------|-------------|
| 名称 | 电商客服 | 产科医生 |
| AI 名称 | 小美 | 小Q |
| 职业 | 客服专员 | 产科医生 |
| 风格 | 热情友好，耐心 | 专业简洁，温和 |
| 规则 | 不承诺非官方优惠 | 不推荐药物 |
| 开场白 | 亲，有什么可以帮您？ | 您好，请告诉我症状 |

### 激活人设

点击人设旁的「启用」按钮，AI 回复时将使用该人设。

## 5. 使用会话中心

点击左侧「会话」。

### 界面说明

- **左侧**：会话列表，显示所有收到消息的对话
- **右侧**：选中会话的聊天记录
- **顶部**：模型快速切换、自动回复模式选择

### 自动回复模式

选中一个会话后，顶部有三个模式按钮：

| 模式 | 说明 |
|------|------|
| **手动** | AI 生成回复但不发送，你手动确认 |
| **半自动** | AI 置信度高时自动发送，不确定时等你确认 |
| **全自动** | AI 全自动回复 |

> 建议先用「手动」观察 AI 回复质量，满意后再切换到「全自动」。

### 群聊说明

- 群里只有被 @提及时才触发 AI 回复
- 每个人在群里的对话独立，多人同时问不会混淆
- 回复会自动 @对应的提问者

## 6. 安全设置

### 锁屏密码

应用支持设置锁屏密码，5 分钟无操作自动锁定。登录失效时自动解锁跳转登录页。

### 数据安全

- 所有对话数据存储在本地 SQLite 数据库
- API Key 加密存储
- 不上传任何数据到第三方服务器

## 7. 常见问题

**Q: 首次启动卡在"下载运行环境"？**
A: 检查网络，Node.js 下载约 60MB，国内网络可能较慢，耐心等待。

**Q: 微信扫码后频繁掉线？**
A: 微信网页协议不稳定，建议用小号。应用有自动重连机制，连续 3 次失败才判定离线。

**Q: AI 回复很慢？**
A: 换更快的模型（如 `deepseek-chat`、`qwen-turbo`、`doubao-pro-4k`）。

**Q: AI 回复不符合预期？**
A: 检查人设配置；调低创意度让回复更稳定；尝试换模型。

**Q: Telegram 连不上？**
A: 需要科学上网。手机号必须带区号（如 `+86`）。

**Q: 支持同时登录多个渠道吗？**
A: 目前每次只能登录一个渠道。

**Q: Ollama 本地模型怎么用？**
A: 安装 Ollama（https://ollama.com），终端运行 `ollama pull qwen2.5`，然后在 AI 设置选择 Ollama 平台，无需 API Key。
